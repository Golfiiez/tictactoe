# tictactoe-vue

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Locally preview the production build
```
npm run preview
```

### Lints and fixes files
```
npm run lint
```

### Customize configuration
This project uses Vite + Vue 3.
